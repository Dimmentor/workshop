# Ollama / OpenWebUI Parameters Reference

## Основные параметры

| Параметр | Тип | Описание | Влияние | Планирование | Программирование | Тестирование / Документация |
|----------|-----|----------|---------|--------------|------------------|-----------------------------|
| temperature | float (0–2) | Контролирует случайность генерации | ↑ креативность / ↓ детерминированность | 0.7–1.0 | 0.1–0.3 | 0.2–0.4 |
| top_p | float (0–1) | Nucleus sampling | ↓ = более строгий выбор | 0.9 | 0.8–0.95 | 0.8–0.95 |
| top_k | int | Ограничение топ-K токенов | ↓ = стабильнее | 40–100 | 20–50 | 20–50 |
| repeat_penalty | float (1.0+) | Штраф за повторения | ↑ = меньше повторов | 1.1–1.2 | 1.05–1.15 | 1.1–1.2 |
| presence_penalty | float | Штраф за повтор тем | ↑ разнообразие | 0.3–0.6 | 0.0–0.2 | 0.2–0.4 |
| frequency_penalty | float | Штраф за частые токены | ↓ повторы слов | 0.2–0.5 | 0.0–0.2 | 0.2–0.4 |
| mirostat | int (0/1/2) | Адаптивная генерация | Контроль перплексии | 2 | 0–1 | 1–2 |
| mirostat_tau | float | Целевая энтропия | ↑ разнообразие | 5–8 | 3–5 | 4–6 |
| mirostat_eta | float | Скорость адаптации | ↑ изменчивость | 0.1–0.2 | 0.05–0.1 | 0.05–0.1 |
| num_ctx | int | Размер контекста | ↑ память | 8k–32k | 16k–128k | 16k–64k |
| num_predict | int | Макс. длина ответа | Ограничение вывода | 500–2000 | 200–1500 | 500–3000 |
| stop | string[] | Стоп-последовательности | Контроль окончания | optional | часто | часто |
| seed | int | Фиксация генерации | Детерминированность | optional | важно | важно |
| tfs_z | float | Tail Free Sampling | ↓ шум | 1.0–2.0 | 1.0 | 1.0–1.5 |
| typical_p | float | Typical sampling | Баланс вероятностей | 0.9–1.0 | 0.8–0.95 | 0.9 |
| repeat_last_n | int | Окно repeat penalty | ↑ контроль повторов | 64–256 | 32–128 | 64–256 |

## Параметры производительности и памяти

| Параметр | Тип | Описание | Рекомендация |
|----------|-----|----------|--------------|
| num_gpu | int | Кол-во GPU слоёв | max |
| num_thread | int | CPU потоки | auto |
| low_vram | bool | Экономия VRAM | off (если можно) |
| f16_kv | bool | FP16 кеш | on |
| use_mmap | bool | mmap загрузка | on |
| use_mlock | bool | Фиксация в RAM | optional |

## Отладка и дополнительные

| Параметр | Тип | Описание | Использование |
|----------|-----|----------|----------------|
| logits_all | bool | Возвращает logits | debugging |
| vocab_only | bool | Только словарь | debugging |
| penalize_newline | bool | Штраф за перенос строки | влияет на форматирование |

# Рекомендованные пресеты

## Планирование (архитектура, дизайн)
```bash
temperature = 0.8
top_p = 0.9
top_k = 50
repeat_penalty = 1.15
mirostat = 2
mirostat_tau = 6
```

## Тестирование
```bash
temperature = 0.2
top_p = 0.9
top_k = 40
repeat_penalty = 1.1
seed = 42
```

## Программирование
```bash
temperature = 0.3
top_p = 0.9
top_k = 40
repeat_penalty = 1.15
```

## Документация
```bash
temperature = 0.4
top_p = 0.95
top_k = 50
repeat_penalty = 1.1
```
